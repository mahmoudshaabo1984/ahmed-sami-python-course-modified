## Course Sections Overview

### Section 1 — Introduction to Python
**Lessons:** 2–6 | **Code:** `codes/section 1/`
- What Python is and why to learn it
- Installing Python and setting up the environment
- Running your first Python script

### Section 2 — Variables, Data Types & Operators
**Lessons:** 10–16 | **Code:** `codes/section 2/`
- Variables and assignment
- Data types: `int`, `float`, `str`, `bool`
- Arithmetic, comparison, and logical operators
- Type conversion

### Section 3 — Strings
**Lessons:** 17–19 | **Code:** `codes/section 3/`
- String creation and operations
- String methods and formatting
- Slicing and indexing

### Section 4 — Lists, Tuples & Sets
**Lessons:** 20–24 | **Code:** `codes/section 4/`
- Lists: creation, indexing, methods
- Tuples: immutability and use cases
- Sets: unique elements, set operations

### Section 5 — Functions
**Lessons:** 25–29 | **Code:** `codes/section 5/`
- Defining and calling functions
- Parameters, return values, default arguments
- Scope: local vs global

### Section 6 — Dictionaries
**Lessons:** 30–32 | **Code:** `codes/section 6/`
- Creating and accessing dictionaries
- Dictionary methods: `.keys()`, `.values()`, `.items()`, `.get()`
- Nested dictionaries

### Section 7 — Loops
**Lessons:** 33–35 | **Code:** `codes/section 7/`
- `while` loop: condition-based repetition
- `for` loop: iterating with `range()` and over lists
- Loop control: `break` and `continue`

### Section 8 — Object-Oriented Programming
**Lessons:** 37–38 | **Code:** `codes/section 8/`
- Classes and objects
- `__init__` constructor and instance attributes
- Instance methods

### Section 9 — pip & Modules
**Lessons:** 40–43 | **Code:** `codes/section 9/`
- PyPI: the Python package index
- Installing and uninstalling packages with `pip`
- Importing built-in modules: `random`, `time`, `datetime`

### Section 10 — Game Development with Turtle
**Lessons:** 45–50 | **Code:** `codes/section 10/`
- Python Turtle graphics
- Building a Ping Pong game step by step

### Section 11 — Final Projects
**Lessons:** 51–53 | **Code:** `codes/section 11/`
- Lesson 51: major project in 5 progressive parts (`51p1.py`–`51p5.py`)
- Lesson 52: tested project variant
- Lesson 53: age calculator & final project (`calculate_age.py`, `calculate_age_final_Project.py`)

## How to Use

1. Navigate to `codes/section N/` for the section you are studying.
2. Open the corresponding notes in `lectures/section N/` and `Udemy notes/section N/`.
3. Run a code file:

```bash
python "codes/section 7/33.py"
```

4. Compare your code against the tested versions (files ending in `test.py`).
