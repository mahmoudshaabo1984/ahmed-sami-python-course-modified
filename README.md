## 🎯 Target Audience

This repository is designed for:
- Blind and visually impaired learners following Ahmed Sami's Python course on Udemy
- Arabic-speaking students who want annotated, tested code alongside lecture notes
- Screen reader users (NVDA on Windows, Orca on Linux) who need plain-text accessible files

---

## 🛠️ How to Use This Repository

1. Navigate to `files/lectures/` and open the section you are studying.
2. Read the `.docx` summary document for that section.
3. Open the corresponding lesson notes in `files/Udemy notes/` for a detailed breakdown.
4. Run the code files from the `codes/` folder:

```bash
python codes/1/2.py
```

5. Compare your code against the tested versions (files ending in `test.py`).
6. For the final project, run:

```bash
python "codes/25/calculate_age_final_Project.py"
```
